# Utility

General-purpose commands for checking bot status and network health.

## `!help`
Displays the standard help menu with all available prefix and slash commands.

**Usage:**
```bash
!help
```

## `!ping`
Check the bot's Discord WebSocket latency and API response time.

**Usage:**
```bash
!ping
```

## `!usage`
View resource usage statistics, memory consumption, and active shard metrics.

**Usage:**
```bash
!usage
```

## `!pagetest`
A developer utility to test the pagination embedding system.

**Usage:**
```bash
!pagetest
```

## `!universe`
Easter egg. Are we alone?

**Usage:**
```bash
!universe
```
