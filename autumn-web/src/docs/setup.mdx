# Setup & Configuration

These commands are used by server administrators to configure Autumn's moderation behavior and access control.

## `!modlogchannel`
Set or update the channel where moderation actions and logs are recorded.

**Usage:**
```bash
!modlogchannel set <#channel>
!modlogchannel clear
```

**Description:**
Requires `Manage Server` permission. Once configured, all warnings, bans, timeouts, and deleted messages will be logged directly to this channel.
